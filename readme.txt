Developer Install
1. Install nodejs
2. Install grunt (sudo npm install -g grunt-cli)
3. Navigate to bugsnap root folder and install modules (npm install)
4. Setup the environment (grunt install)
5. Package extension (grunt)
6. D'n'D resulting *.crx file to Extensions page in Chrome

Known issues:
Chrome won't load unpacked extension on Mac OS X due to symlinks presence.
