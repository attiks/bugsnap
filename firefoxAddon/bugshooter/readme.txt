To get FF addon work:

1. add pointer file to your FF extensions dir (C:\Users\[username]\AppData\Roaming\Mozilla\Firefox\Profiles\[profilename]\extensions)
2. name of this file should match to the value of <em:id> field of the install.rdf file (bugshooter@agetes.com)
3. put the real path to your addon as content of pointer file
4. restart Firefox
